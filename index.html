A1 · CREDIT

Credit Risk & Spreads

Selisih yield antara obligasi korporasi dan treasury mencerminkan kompensasi atas risiko gagal bayar (default risk).

Credit Spread
\[ \text{Spread} = y_{\text{corporate}} - y_{\text{treasury}} \]
Default Probability (aproksimasi)
\[ P(\text{default}) \approx \frac{\text{Spread}}{1 - \text{Recovery Rate}} \]

Recovery rate biasanya 30–40% untuk senior debt.

Z-Spread
\[ P = \sum_{t} \frac{CF_t}{(1 + r_t + z)^t} \]

② Estimasi Probabilitas Default

③ Harga Obligasi dengan Z-Spread

Multiples
\[ P/E = \frac{\text{Price}}{\text{EPS}}, \quad P/B = \frac{\text{Price}}{\text{Book Value}}, \quad EV/EBITDA = \frac{EV}{EBITDA} \]

A3 · RATIOS

Key Financial Ratios

Rasio keuangan inti untuk membaca profitabilitas, likuiditas, leverage, dan efisiensi perusahaan — plus dekomposisi DuPont untuk ROE.

Profitabilitas
\[ ROE = \frac{\text{Net Income}}{\text{Equity}}, \quad ROA = \frac{\text{Net Income}}{\text{Assets}}, \quad \text{Profit Margin} = \frac{\text{Net Income}}{\text{Revenue}} \]
Likuiditas
\[ \text{Current Ratio} = \frac{CA}{CL}, \quad \text{Quick Ratio} = \frac{CA - \text{Inventory}}{CL} \]
Leverage
\[ D/E = \frac{\text{Debt}}{\text{Equity}}, \quad \text{Interest Coverage} = \frac{EBIT}{\text{Interest}} \]
Efisiensi
\[ \text{Asset Turnover} = \frac{\text{Sales}}{\text{Assets}}, \quad \text{Inventory Turnover} = \frac{COGS}{\text{Avg Inventory}} \]
DuPont Decomposition
\[ ROE = \frac{NI}{\text{Sales}} \times \frac{\text{Sales}}{\text{Assets}} \times \frac{\text{Assets}}{\text{Equity}} \]

B1 · REGRESSION

Regression Analysis

Masukkan dua deret data (X, Y) dan kalkulator akan menghitung estimator OLS serta R² secara langsung.

Simple Linear Regression
\[ Y = \alpha + \beta X + \epsilon \]
OLS Estimators
\[ \beta = \frac{\text{Cov}(X,Y)}{\text{Var}(X)}, \quad \alpha = \bar{Y} - \beta \bar{X} \]
Multiple Regression (Matrix Form)
\[ Y = \beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k + \epsilon, \qquad \hat{\beta} = (X'X)^{-1}X'Y \]
R² (Goodness of Fit)
\[ R^2 = 1 - \frac{SS_{\text{res}}}{SS_{\text{tot}}} \]

① OLS dari Data (X, Y)

B2 · HYPOTHESIS

Hypothesis Testing

Uji signifikansi rata-rata sampel terhadap nilai hipotesis menggunakan t-statistic.

t-test (perbandingan rata-rata)
\[ t = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \]
F-test

Digunakan untuk membandingkan varians atau menguji signifikansi keseluruhan model regresi.

Error Tipe I & II

Tipe I: P(Tipe I) = α — menolak H₀ padahal benar (false positive). Tipe II: gagal menolak H₀ padahal salah (false negative).

① t-statistic

Distribusi Normal

\[ \bar{X}_n \xrightarrow{d} N\left(\mu, \frac{\sigma^2}{n}\right) \]

Skewness & Kurtosis
\[ \gamma = \frac{E[(X-\mu)^3]}{\sigma^3}, \qquad \kappa = \frac{E[(X-\mu)^4]}{\sigma^4} \]

B5 · CORRELATION

Correlation & Covariance

Ukuran hubungan linear antar dua variabel — dasar dari diversifikasi portofolio pada bagian MPT.

Covariance
\[ \text{Cov}(X,Y) = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - E[X]E[Y] \]
Correlation
\[ \rho_{X,Y} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y} \in [-1, 1] \]
Correlation Matrix
\[ \Sigma_{ij} = \rho_{ij}\sigma_i \sigma_j \]

C1 · TVM

Time Value of Money

Fondasi seluruh valuasi keuangan: nilai uang hari ini berbeda dengan nilai uang di masa depan.

Present Value (diskrit & kontinu)
\[ PV = \frac{FV}{(1+r)^n} \qquad PV = FV \cdot e^{-rt} \]
Future Value (diskrit & kontinu)
\[ FV = PV(1+r)^n \qquad FV = PV \cdot e^{rt} \]
PV Annuity
\[ PV_{\text{annuity}} = C \cdot \frac{1-(1+r)^{-n}}{r} \]
PV Perpetuity
\[ PV_{\text{perpetuity}} = \frac{C}{r} \]
Effective Annual Rate
\[ r_{\text{eff}} = \left(1+\frac{r}{m}\right)^m - 1 \]

C2 · BONDS

Fixed Income: Bonds

Harga obligasi, yield, dan sensitivitas harga terhadap perubahan suku bunga (duration & convexity).

Bond Price
\[ P = \sum_{t=1}^n \frac{C}{(1+r)^t} + \frac{F}{(1+r)^n} \]
Current Yield
\[ \text{Current Yield} = \frac{\text{Annual Coupon}}{\text{Price}} \]
Macaulay Duration
\[ D = \frac{\sum_{t=1}^n t \cdot PV(CF_t)}{P} \]
Modified Duration
\[ D^{*} = \frac{D}{1+y}, \qquad \frac{\Delta P}{P} \approx -D^{*}\Delta y \]
Convexity
\[ C_{vx} = \frac{1}{P}\frac{\partial^2 P}{\partial y^2} \]

D1 · MPT

Diversifikasi menurunkan risiko portofolio selama korelasi antar aset tidak sempurna positif.

Expected Return Portfolio
\[ E[R_p] = \sum_{i=1}^n w_i E[R_i] \]

\[ \sigma_p^2 = w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\sigma_1\sigma_2\rho_{12} \]
Portfolio Variance (umum)
\[ \sigma_p^2 = \sum_{i=1}^n\sum_{j=1}^n w_i w_j \sigma_i \sigma_j \rho_{ij} \]

D2 · CAPM

Capital Asset Pricing Model

Menghubungkan return yang diharapkan suatu aset dengan risiko sistematisnya (beta) relatif terhadap pasar.

CAPM
\[ E[R_i] = R_f + \beta_i(E[R_m]-R_f) \]
Beta
\[ \beta_i = \frac{\text{Cov}(R_i,R_m)}{\text{Var}(R_m)} = \rho_{i,m}\frac{\sigma_i}{\sigma_m} \]
Alpha (Jensen's Alpha)
\[ \alpha_i = R_i - [R_f + \beta_i(R_m-R_f)] \]

E3 · STOCHASTIC

Stochastic Calculus Basics

Brownian motion, Geometric Brownian Motion (model dasar harga saham), dan Itô's Lemma.

Brownian Motion W_t

· W₀ = 0    · Wₜ − Wₛ ∼ N(0, t−s)    · (dWₜ)² = dt

Geometric Brownian Motion
\[ dS_t = \mu S_t\, dt + \sigma S_t\, dW_t \]
Solusi GBM
\[ S_t = S_0 \exp\left[\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right] \]
Itô's Lemma — untuk Y = f(S_t, t)
\[ dY = \left(\frac{\partial f}{\partial t} + \mu S_t \frac{\partial f}{\partial S} + \frac{1}{2}\sigma^2 S_t^2 \frac{\partial^2 f}{\partial S^2}\right)dt + \sigma S_t \frac{\partial f}{\partial S}\,dW \]

Contoh: jika Y = ln(Sₜ), maka dY = (μ − ½σ²)dt + σ dWₜ — inilah asal rumus solusi GBM di atas.
